## Interaction States

### Captain Card States

**Default State:**

- Border: `border-neutral-200`
- Background: `bg-white`
- Cursor: default

**Hover State:**

- Border: `border-[#EC2227]/50`
- Shadow: `shadow-lg`
- Opacity: arrow appears
- Transform: subtle scale

**Click State:**

- Modal opens
- Backdrop appears
- Smooth transition

### Modal States

**Closed:**

- Hidden (display: none)
- No backdrop

**Opening:**

- Backdrop fades in
- Modal slides up
- ~200ms animation

**Open:**

- Fully visible
- Interactive
- Can scroll if needed

**Closing:**

- Reverse animation
- 200ms fade out
- Body scroll restored
